Microservico creado en JAVA SPRINGBOOT usando endpoints del tipo GET.
El cual nos permitirá ver los detalles de envio o envios y las ubicaciones.

para usarlo, usar las siguientes url:

http://localhost:8080/envio
Permite mostrar todos los envíos y sus detalles

http://localhost:8080/envio/1
Permite mostrar un envió y sus detalles

http://localhost:8080/envio/ubicacion
Permite mostrar la ubicación de los envíos

http://localhost:8080/envio/ubicacion/1
Permite mostrar la ubicación de un envió
